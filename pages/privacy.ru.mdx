## Какие данные собирает и хранит этот сайт?
Сайт собирает и хранит данные, которые необходимы для его корректного функционирования.

## Какие данные собирает Civ6Bot?
При использовании ботов Civ6Bot (далее - бот) на сервере Discord (далее - сервер), бот собирает следующую информацию о сервере:
- название сервера, ID сервера;
- количество участников на сервере.

При использовании ботов участниками сервера (далее - пользователи), бот собирает следующую информацию:
- никнейм пользователя, ID пользователя;
- данные, которые были получены с разрешения пользователя или предоставлены самим пользователем:
    - ID аккаунта Steam;
    - данные о рейтинговых играх.

## Как используются мои данные?
Предоставленные пользователями данные собираются и используются для следующих целей:
- реализация заявленных возможностей ботов (хранение данных и возможность получения информации участниками);
- анализ полученных данных для выявления проблем и дальнейшей разработки.

## Где и как публикуются мои данные?
Предоставленные вами данные могут полученами пользователями сервера, на котором вы предоставили свои данные, перечисленные выше.

Предоставленные вами данные могут быть предоставлены в обезличенном виде третьим лицам с целью получения игровой статистики игровыми сообществами и/или для улучшения качества игрового контента.

Информация об ID аккаунтов Steam не подлежит публикации.
